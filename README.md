# side-projects

A collection of side projects built while studying at IE Business School.

---

## Projects

| Folder | Project | Status | Description |
|--------|---------|--------|-------------|
| [`student-food-marketplace/`](./student-food-marketplace) | Campus Bites | In progress | Home-cooked food marketplace for IE students — browse, reserve, and pick up meals at the cafeteria |
| [`job-radar/`](./job-radar) | JobRadar | Code exists locally | Auto-track job applications from your inbox — no more spreadsheets |
| [`lifelong-learning/`](./lifelong-learning) | Lifelong Learning | Spec stage | IMBA course content hub that replaces Blackboard — fast, searchable, mobile-first |
| [`ie-student-services/`](./ie-student-services) | IE Student Services | Spec stage | Peer marketplace where current students help incoming students settle in Madrid — housing, tutoring, knowledge handoff |

---

## Stack

All projects use the same core stack: **Next.js · Supabase · Drizzle ORM · TypeScript**
